You are applying the visual language for Teseo, an AI SaaS for TCC creation.

Goals:
- move the product away from generic old SaaS styling
- make the interface feel calm, premium, editorial, and trustworthy
- align dashboard, onboarding, workspace, modals, and buttons into one consistent visual system

Design direction:
- inspired by Claude / Anthropic visual language
- warm neutral surfaces instead of cold white or dark neon
- terracotta / clay accent instead of blue or purple
- editorial, intelligent, minimal, calm
- strong spacing, low visual noise
- no flashy gradients, glassmorphism, neon glow, or gamer aesthetics

Color palette:
- background main: #faf9f5
- background subtle: #f3f1ea
- surface: #ffffff
- surface warm: #f6f3ed
- border: #e7e1d6
- text primary: #171717
- text secondary: #6b665f
- accent primary: #c4663a
- accent hover: #b85b31
- accent soft: #e9d2c6
- success: #5f7a65
- warning: #ad7a2f
- danger: #b4544d
- dark background option: #141413
- dark text on dark backgrounds: #faf9f5

Typography:
- use a refined editorial feel
- headings should feel confident and slightly literary
- body text should be clean, readable, calm
- prefer strong hierarchy over decorative effects
- use generous line-height when text density is high

Component rules:
- buttons should have clear hierarchy: primary, secondary, ghost
- primary buttons use accent primary
- secondary buttons use warm neutral surfaces and borders
- cards should feel soft and premium, not sterile
- borders should be subtle
- radius should be medium and consistent
- shadows should be light and used sparingly
- badges should be compact and elegant
- inputs should feel calm and premium, not overly outlined

Layout rules:
- prioritize whitespace
- reduce clutter
- remove unnecessary separators and boxes
- keep UI breathable
- prefer fewer, more meaningful actions
- keep dashboard and workspace visually coherent

When refactoring existing UI:
- inspect the existing components first
- preserve functionality unless explicitly asked to change it
- update old colors, old button styles, inconsistent spacing, and generic cards
- reduce visual noise before adding anything new

Apply this design system to:
- dashboard
- create TCC onboarding
- workspace
- header
- sid
